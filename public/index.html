The Fantasy API

This API currently serves data on Harry Potter & Star Wars characters and movies. Harry Potter Data from Gulsah Demiryurek on kaggle, Star Wars Data from Joe Young on kaggle.

Routes & Resources

  • Base Url: https://the-fantasy-api.herokuapp.com
  • /api/v1/movies/harry-potter/ - Get All Harry Potter Movies
  • /api/v1/movies/harry-potter/:id - Get Harry Potter Movie by Id
  • /api/v1/movies/harry-potter?q=search_term - Search for a Harry Potter Movie by title

  • /api/v1/characters/harry-potter/ - Get All Harry Potter Characters
  • /api/v1/characters/harry-potter/:id - Get Harry Potter Character by Id
  • /api/v1/characters/harry-potter?q=search_term - Search for a Harry Potter Character by name

  • /api/v1/api/v1/movies/star-wars/ - Get All Star Wars Movies
  • /api/v1/api/v1/movies/star-wars/:id - Get Star Wars Movie by Id
  • /api/v1/api/v1/movies/star-wars?q=search_term - Search for a Star Wars Movie by title

  • /api/v1/api/v1/characters/star-wars/ - Get All Star Wars Characters
  • /api/v1/api/v1/characters/star-wars/:id - Get Star Wars Character by Id
  • /api/v1/api/v1/characters/star-wars?q=search_term - Search for a Star Wars Character by name
