The Fantasy API

This API currently serves data on Harry Potter & Star Wars characters and movies. Harry Potter Data from Gulsah Demiryurek on Kaggle, Star Wars Data from Joe Young on Kaggle.

Getting Started

This API is free to use, and no API key is required. Please be reasonable with your usage :)

All example code is written in JavaScript using the Axios HTTP client package.


Routes & Resources

Movies

The 'Movies' resource provides data on each movie related to the specified franchise. Data for specific movies can also be accessed by either specifying the movie's id or adding search parameters to your request (searching the movie by title).
Includes data on: title, overview and release_date.

Harry Potter

This resource includes all data for the main 8 Harry Potter movies. Ids start at 1 [Philosopher's Stone] and end at 8 [The Deathly Hallows: Part 2].

Star Wars

This resource includes all data for the main 9 Star Wars movies, as well as Solo and Rogue One. Ids start at 15 [A New Hope], and end at 25 [Solo].


Characters

The 'Characters' resource provides data on *almost all* characters related to the specified franchise. Data for specific characters can also be accessed by either specifying the character's id or adding search parameters to your request (searching a character by name).

Harry Potter

This resource includes all data for ~140 Harry Potter characters. Ids start at 1 [Harry James Potter], and end at 140 [Kreacher].

Includes data on:

  • name
  • gender
  • job
  • house
  • wand
  • patronus
  • species
  • blood_status
  • hair_colour
  • eye_colour
  • loyalty
  • skills
  • birth_year
  • death_year
